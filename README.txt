ШАХМАТКА — ДЕМО-СБОРКА (статическая, без бэкенда)
==================================================
Открыть локально:  просто открой index.html в браузере.
Захостить:         залей ВСЕ файлы из этой папки в GitHub-репозиторий →
                   Settings → Pages → Branch: main /(root). Ссылка готова.

Что внутри:
- Данные вшиты в demo-data.js (снимок твоей таблицы 1:1, формулы не нужны).
- Бэкенд не вызывается: правки % живут в памяти, сбрасываются кнопкой «Сбросить» или перезагрузкой.
- Три роли-чипсины справа: Аноним (смотрит) · Подрядчик (правит %, без контрактов) ·
  Генподрядчик (всё + Сводка/Подготовка). Пароль убран.
- Прод-URL Apps Script и Code.gs в сборку НЕ включены.

Аналитику (счётчик/гео/записи сессий) можно вставить тегом в index.html — см. переписку.
